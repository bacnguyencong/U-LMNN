U-LMNN - Distance metric learning with the Universum

U-LMNN contains the implementation of Large Margin Nearest Neighbor (LMNN) with the Universum. Our implementation is baed on the implementation of LMNN (http://www.cs.cornell.edu/~kilian/code/code.html)
